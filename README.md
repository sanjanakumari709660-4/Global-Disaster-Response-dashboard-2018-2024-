🌍 Global Disaster Response Dashboard
📊 Project Overview

The Global Disaster Response Dashboard is a Power BI project that analyzes global disaster data from 2018 to 2024.
This dashboard helps to understand the frequency, impact, and economic consequences of disasters across different countries.

The purpose of this project is to transform raw disaster data into clear and interactive visual insights that support better understanding of disaster patterns worldwide.

🎯 Objective

The main objective of this project is to:

Analyze global disaster trends.

Understand the impact of disasters on human lives and the economy.

Provide clear visual insights using interactive Power BI dashboards.

🛠 Tools & Technologies

Power BI – Data visualization and dashboard creation

Microsoft Excel / CSV Dataset – Data source

Data Analysis Techniques – For extracting insights from disaster data

📌 Key Features of the Dashboard

Overview of total disasters worldwide

Analysis of total casualties caused by disasters

Visualization of economic losses

Country-wise disaster analysis

Disaster type distribution

Interactive filters and slicers for better exploration of data

By visualizing disaster data, this project helps in better understanding disaster trends and improving awareness for disaster response planning.
